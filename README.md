# corgi-capital.com

Website for [Corgi Capital](https://corgi-capital.com).

## Stack

Static HTML/CSS/JS. No build step. Hosted via GitHub Pages with custom domain.

## Local dev

```bash
open index.html
# or
python3 -m http.server 8080
```

## Deployment

Push to `main`. GitHub Pages deploys automatically.
